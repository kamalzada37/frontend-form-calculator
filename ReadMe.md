# Frontend Form Calculator

This project is a simple interactive web application built for a frontend web development assignment during my bachelor's studies. It uses HTML, CSS, Bootstrap, and JavaScript to calculate a final value based on user-selected form inputs.

## Project Overview

The application demonstrates:

- Form design using HTML
- Styling with CSS and Bootstrap
- User interaction with JavaScript
- Dynamic price calculation based on multiple conditions
- DOM manipulation and event handling

## Features

- Dropdown selections for education, family net worth, and caste
- Checkbox inputs for skills and reputation factors
- Radio button input for age category
- Automatic final price calculation
- Clean and responsive layout using Bootstrap

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- JavaScript
